Componente Descripción
Integración de datos Procesos que permiten extraer, transformar y consolidar información proveniente de múltiples sistemas empresariales.
Almacenamiento analítico Repositorios especializados como Data Warehouse o Data Mart que almacenan información histórica para análisis.
Herramientas analíticas Plataformas que permiten explorar los datos mediante análisis multidimensional, minería de datos o análisis estadístico.
Visualización de datos Sistemas de reportes y paneles de control que facilitan la interpretación de la información.
